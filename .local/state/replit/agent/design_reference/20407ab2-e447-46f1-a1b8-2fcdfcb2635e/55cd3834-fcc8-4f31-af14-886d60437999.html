"Why was JSX introduced to React?"

Key Insight

JSX was introduced in early React development (commits a1b2c3d through e4f5g6h) to solve the problem of complex, nested function calls for creating elements. The team wanted a more declarative syntax that resembled HTML while maintaining JavaScript's power.

Add initial JSX transformer and syntax support

AI Analysis

This commit introduces JSX (JavaScript XML) syntax to React, allowing developers to write HTML-like syntax within JavaScript. The change was motivated by the need to make React components more readable and maintainable compared to verbose React.createElement calls.
